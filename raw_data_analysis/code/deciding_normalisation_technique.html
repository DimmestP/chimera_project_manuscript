Analysis of RT-qPCR data sets with two experimental replicates under different normalisation

Sam Haynes

22/07/2020

  • Comparison of transcript abundance linear models

Comparing the normalistion of four qPCR datasets; pRPS3-tRPS3, pPGK1-tRPS3, pTSA1-tTSA1 and pPGK1-tTSA1, by three different normalisation genes; RPS3, PGK1, URA3. First only one normalisation gene is used for each construct and then the mean/median of all genes is used.

collated_norm_mean_data <- pPGK1_tRPS3_norm_mean %>%
  bind_rows(pRPS3_tRPS3_norm_mean) %>%
  bind_rows(pPGK1_tTSA1_norm_mean) %>%
  bind_rows(pTSA1_tTSA1_norm_mean)

collated_norm_median_data <- pPGK1_tRPS3_norm_median %>%
  bind_rows(pRPS3_tRPS3_norm_median) %>%
  bind_rows(pPGK1_tTSA1_norm_median) %>%
  bind_rows(pTSA1_tTSA1_norm_median)

collated_norm_PGK1_data <- pPGK1_tRPS3_norm_PGK1 %>%
  bind_rows(pRPS3_tRPS3_norm_PGK1) %>%
  bind_rows(pPGK1_tTSA1_norm_PGK1) %>%
  bind_rows(pTSA1_tTSA1_norm_PGK1)

collated_norm_RPS3_data <- pPGK1_tRPS3_norm_RPS3 %>%
  bind_rows(pRPS3_tRPS3_norm_RPS3) %>%
  bind_rows(pPGK1_tTSA1_norm_RPS3) %>%
  bind_rows(pTSA1_tTSA1_norm_RPS3)

collated_norm_URA3_data <- pPGK1_tRPS3_norm_URA3 %>%
  bind_rows(pRPS3_tRPS3_norm_URA3) %>%
  bind_rows(pPGK1_tTSA1_norm_URA3) %>%
  bind_rows(pTSA1_tTSA1_norm_URA3)

Comparison of transcript abundance linear models
